## Modifications
This modified version ("ECHO CORP Blackbox Viewer") includes the following changes:

- **Visual Redesign:** Complete UI overhaul with cyberpunk/neon theme including:
  - Custom color scheme (cyan/purple neon palette)
  - Glassmorphism effects and backdrop blur styling
  - CRT scanline and vignette overlays
  - Redesigned welcome page with hero section
  - Custom button, panel, and control styling
  
- **Branding:** Changed application name and branding from "Betaflight" to "ECHO CORP"

- **Feature Additions:**
  - One-click CSV conversion on welcome page
  - Enhanced home page with quick-action cards
  - Feature showcase section

- **Code Modifications:**
  - Added CSV conversion handler in `src/main.js`
  - Modified graph colors in `src/grapher.js`
  - Updated timeline colors in `src/seekbar.js`
  - Extensive CSS additions in `src/css/main.css` (ECHO CORP theme section)
  - Updated HTML structure in `index.html`
